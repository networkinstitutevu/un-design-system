# Stats card slider

This is collection of [Stats cards](/docs/components-ui-components-cards-stats-card--stats-card) organized in slider

<Canvas>
  <Story name="Stats card slider">
    {(args, { globals: { locale, accent } }) => {
      const caption = statsArray(locale);
      return <Statscardslider data={caption} {...args}></Statscardslider>;
    }}
  </Story>
</Canvas>

### Usage

- Copy HTML from the HTML tab of canvas and also include the css and Js files listed below. Remove all inline styling (added by Swiper)
- Include the CSS and JS resource defined in the [Stats card](/docs/components-ui-components-cards-stats-card--stats-card#css-and-js-references). Pay attention to any JS initiation required for them.
- include the CSS and JS files listed below.
- Initiliase swiper calling following function: `swiper('.stats-slider');`
- Refer [this document](https://github.com/undp/design-system/wiki/Swiper-documentation) for Swiper integration & options
