# Progress bar

The Progress Bar provides a visual representation of a user’s progress on-page.

### Overview

A progress bar is a graphical content element that shows the progress of an ongoing task on a computer.

#### When to use:

When the process requires more time to complete the user gets the exact progress.

### Formatting

#### Default

The progress bar is an elongated rectangular bar that displays the progress of an extended computer operation.

### Behaviors

#### States

The progress bar has two states: default and short.

The following behavior modifiers are available:

Progress Bar 0:
Progressbar provides a visual representation of a user’s progress on a page indicating the progress bar is in 0 state showing the progress bar is in the initial state.

Progress Bar 25:
Progressbar provides a visual representation of a user’s progress on a page indicating the progress bar is in 25% progress.

Progress Bar 50:
Progressbar provides a visual representation of a user’s progress on a page indicating the progress bar is in 50% progress.

Progress Bar 100:
Progressbar provides a visual representation of a user’s progress on a page indicating the progress bar is in 100% progress.
