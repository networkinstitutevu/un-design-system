# Parallax Gallery

The Parallax Gallery is a trend where the images move at a different speed than the content.

### Overview

The Parallax Gallery component is created for showing gallery, designation, and image in a particular section. It is a multi-usage component that creates teasing kinds of content.

### When to use:

To create engaging and interactive content.

### Formatting

#### Default

It consists of a gallery of images, description,s and CTA links.

### Content

Parallax Gallery has Default or Gray backgrounds.

### Behaviors

#### States

There are two states: Default and Mobile.

### Usage:

- Copy the HTML from the HTML Tab of canvas and include CSS and JS in ‘CSS and JS References’ section.
- Initialize the parallaxEffect by calling parallaxEffect('.parallax-gallery-images', '.column', 'top center', 'bottom+=15% center', 'vertical', 'all'); function in your JS file.
- Refer [this document](https://github.com/undp/design-system/wiki/Parallax-documentation) for Parallax integration & options

### CSS and JS References

#### CSS:

Add the base layout style from

- https://cdn.jsdelivr.net/gh/undp/design-system/docs/css/base-minimal.min.css
- https://cdn.jsdelivr.net/gh/undp/design-system/docs/css/components/parallax-gallery.min.css
- https://cdn.jsdelivr.net/gh/undp/design-system/docs/css/components/cta-link.min.css

Following external libraries needs to be included first:

- GSAP core library
- GSAP ScrollTrigger plugin
- GSAP EasePack plugin

#### JS:

- https://cdn.jsdelivr.net/gh/undp/design-system/docs/js/parallax.min.js

On the opening of the page and scrolling, images move.
